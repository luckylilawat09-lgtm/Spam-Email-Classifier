# 📧 Spam Email Classifier

A beginner-friendly AI/ML project that classifies email messages as **Spam** or **Not Spam (Ham)** using Machine Learning.

---

## 🚀 Features
- Classifies text messages into Spam or Ham
- Uses **Naive Bayes Algorithm**
- Simple and easy-to-understand code
- Allows real-time user input testing

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- Scikit-learn
